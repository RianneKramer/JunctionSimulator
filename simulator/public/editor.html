How to use:
1. Click a path button to select it
2. Click on the canvas to add waypoints (in order: spawn -> approach -> detect -> stop -> intersection -> exit)
3. Drag existing waypoints to reposition
4. Select a waypoint, then click "Set Detect" or "Set Stop" to mark it
5. Right-click a waypoint to delete it
6. Export when done, paste into simulator HTML

Colors: Blue=North, Red=East, Green=South, Orange=West
Yellow circle = detect zone start, Red circle = stop line
