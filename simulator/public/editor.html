How to use:
1. Add/select a traffic light
2. Add/select a variant under that light
3. Click canvas to add waypoints, drag to move, right-click to delete
4. Set detect and stop indexes per variant
5. Export the resulting `RAW_PATHS` structure

This editor supports frontend-only multi-ending paths per controller signal.
